# dapp-kit-core

A framework-agnostic toolkit for building decentralized applications on [MySo](https://mysocial.network).

`@socialproof/dapp-kit-core` provides an action-based API for wallet operations, automatic state
management via [nanostores](https://github.com/nanostores/nanostores), and built-in
[Web Components](https://developer.mozilla.org/en-US/docs/Web/API/Web_components) for wallet
connection UI. It can be used standalone with vanilla JavaScript, or as the foundation for
framework-specific bindings like
[`@socialproof/dapp-kit-react`](https://www.npmjs.com/package/@socialproof/dapp-kit-react).

## Installation

```sh
npm i @socialproof/dapp-kit-core @socialproof/myso
```

## Documentation

See the [dApp Kit docs](https://sdk.mystenlabs.com/myso/dapp-kit) for full usage instructions,
guides, and API reference.
