# dapp-kit-react

React hooks and components for building decentralized applications on the [MySo](https://mysocial.network)
blockchain. Built on top of
[`@socialproof/dapp-kit-core`](https://www.npmjs.com/package/@socialproof/dapp-kit-core).

Provides a `DAppKitProvider`, ready-to-use `ConnectButton` and `ConnectModal` components, and hooks
like `useDAppKit`, `useCurrentAccount`, `useCurrentWallet`, and more.

## Installation

```sh
npm i @socialproof/dapp-kit-react @socialproof/myso
```

## Documentation

See the [dApp Kit docs](https://sdk.mystenlabs.com/myso/dapp-kit) for full usage instructions,
guides, and API reference.
