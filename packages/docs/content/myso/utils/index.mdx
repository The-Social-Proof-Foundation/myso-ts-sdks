---
title: The `@socialproof/myso/utils` package
---

This package contains some utilities that simplify common operations when working with the MySo
TypeScript SDK.

## Constants

A set of constants exported for common uses cases:

- `MIST_PER_MYSO`: The conversion rate for MIST to MYSO (1,000,000,000)
- `MYSO_DECIMALS`: the number of decimals you must shift a MIST value to convert it to MYSO (`9`)
- `MYSO_ADDRESS_LENGTH`: The number of bytes in a MySo address (32)
- `MOVE_STDLIB_ADDRESS`: The address for the MySo Move standard library
- `MYSO_FRAMEWORK_ADDRESS`: The address for the MySo Framework
- `MYSO_SYSTEM_ADDRESS`: The address for the MySo System module
- `MYSO_CLOCK_OBJECT_ID`: The address for the `myso::clock::Clock` object
- `MYSO_SYSTEM_STATE_OBJECT_ID`: The address for the `MySoSystemState` object
- `MYSO_RANDOM_OBJECT_ID`: The address for the `0x2::random::Random` object

## Formatters

You can use the following helpers to format various values:

- `formatAddress`
- `formatDigest`
- `normalizeStructTag`
- `normalizeMySoAddress`
- `normalizeMySoObjectId`
- `normalizeMySoNSName`
- `normalizeMySoNSName`

## Validators

You can use the following helpers to validate the format of various values (this only validates that
the value is in the correct format, but does not validate the value is valid for a specific use
case, or exists on chain).

- `isValidMySoAddress`
- `isValidMySoObjectId`
- `isValidTransactionDigest`
- `isValidMySoNSName`

## Encoding

The following methods are re-exported to help with converting between commonly used encodings

- `fromHex`: Deserializes a hex string to a Uint8Array
- `toHex`: Serializes a Uint8Array to a hex string
- `fromBase64`: Deserializes a base64 string to a Uint8Array
- `toBase64`: Serializes a Uint8Array to a base64 string
