---
title: zkSend SDK
---

The zkSend SDK provides tools for interacting with the zkSend primitive. It provides functionality
to create your own zkSend Claim Links, with support for any publicly transferrable asset.

## Installation

```sh npm2yarn
npm i @socialproof/zksend @socialproof/myso
```

## Setup

To use the zkSend SDK, create a MySo client and extend it with the zkSend extension:

```ts
import { MySoGrpcClient } from '@socialproof/myso/grpc';
import { zksend } from '@socialproof/zksend';

const client = new MySoGrpcClient({
	network: 'mainnet',
	baseUrl: 'https://fullnode.mainnet.mysocial.network:443',
}).$extend(zksend());
```

The zkSend SDK supports mainnet and testnet networks. The extension automatically configures the
correct contract IDs based on the client's network.
